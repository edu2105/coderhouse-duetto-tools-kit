🧰 Welcome to Duetto Tools Kit

This site was designed by the Integrations team in order to provide you different solutions for your daily work.

Security

Security is an important asset for any company. Since Duetto works essentially with third-party data, this aspect becomes more important. So for the time you need to inspect, convert o manipulate some data, instead of using a public website which you don't know what they are doing with the information, best use the Duetto Tools Kit, and if you cannot find here what you need, then let us know here and suggest a new tool so we can add it.

Reliability

Using an internal company site makes the data manipulation more secure and also can be collaborative since new tools for different departments can be added, no matter the difficulty.

Available Tools

Visit the Tools section to start using them!

The HTNG Validator is a tool that will compare any HTNG message type (such as Reservations, Inventory, Blocks, etc) agains an schema based on Duetto HTNG implementation

The XML To CSV converter will let you transform an XML payload to a CSV file that you can copy or download.

The Rate Sum tool will make the rate calculation for you. You just need to upload any booking stay-day file.
